# 📦 Produktspeicher_erweitert (Vibe Coding)
**Ein funktionales Web-Projekt zur Verwaltung von Produktdaten, erstellt mit React und TypeScript.**

## 📝 Beschreibung
> Dieses Projekt dient zur einfachen Erfassung und Speicherung von Produkten. Es wurde als Vibe-Coding-Projekt umgesetzt.

---

## ✨ Highlights & Funktionen

- 📦 **Produkt-Management**: Vollständiges CRUD (Erstellen, Lesen, Aktualisieren, Löschen) mit Details wie Preis-Tracking, Kategorien und Shop-Links.
- 🧩 **Intelligente Bundles**: Organisiere Produkte in Collections (z.B. Gaming-Setups oder Arbeitsplatz-Ausstattungen).
- 🔍 **Katalog & Smart-Filter**: Blitzschnelle Suche und Filterung nach Haupt- und Unterkategorien.
- 💰 **Budget-Tracking**: Setze monatliche Limits und behalte deine Ausgaben visuell unter Kontrolle.
- 📊 **Analytics-Dashboard**: Detaillierte Statistiken über Produktausgaben, Preisverteilung und Kategorien.
- 💖 **Favoriten-System**: Markiere deine Must-Haves für den schnellen Zugriff.
- 🔔 **Deals & Preisalarme**: Behalte reduzierte Artikel und Preisänderungen direkt im Blick.
- 🛒 **Shop-Verzeichnis**: Verwalte deine liebsten Webshops und kategorisiere sie für effizientes Shopping.
- 🎨 **Theme Creator**: Passe das UI mit dem integrierten Editor an – inklusive **Glassmorphismus (Glass Mode)** und Live-Vorschau.
- 💾 **Daten-Persistenz**: Automatische Echtzeit-Synchronisation mit the LocalStorage deines Browsers.
- 📱 **Premium UI/UX**: Vollständig responsives Design mit flüssigen Animationen und moderner Glas-Optik.

## 🛠️ Tech-Stack

| Technologie | Nutzung |
| :--- | :--- |
| **React 19** | Core Frontend Framework |
| **TypeScript** | Typensicherheit & Stabilität |
| **Tailwind CSS 4** | Modernes Styling & JIT Engine |
| **Zustand** | State Management & Persistence |
| **React Router 7** | Client-seitiges Routing |
| **Lucide React** | Hochwertige Icon-Library |
| **Vite** | Ultra-schneller Build-Prozess |

## 🚀 Installation & Start

1. **Repository klonen**
   ```bash
   git clone https://github.com/Fabian-Kolb/Produktspeicher_erweitert.git
   ```

2. **Abhängigkeiten installieren**
   ```bash
   npm install
   ```

3. **Projekt starten**
   ```bash
   npm run dev
   ```
